# 企业级 AI 知识库系统 — 技术详情

## 产品定位
企业级 AI 知识库系统源码，多智能体 + 多租户 RAG + 可视化工作流 + MCP 接入。可直接部署、可二次开发、可做企业交付，非 Demo 壳子。

## 双版本

| 版本 | 适用对象 | 特点 |
|------|---------|------|
| **服务商版** | 服务商、软件公司、实施公司、私有化交付团队 | 多租户架构，适合项目交付、SaaS 运营、代运营服务。支持平台级管理、租户隔离、品牌隔离、知识库隔离、模型配置隔离、工作流隔离 |
| **企业版** | 个人学习、企业自用、内部知识助手 | 单后台结构，部署更轻，买回去即可直接改品牌、配模型、配知识库、配工作流 |

## 核心功能

1. **多智能体架构** — 每个租户可配置多个智能体，每个智能体独立设置：欢迎语、Prompt、模型、知识范围、工具范围、MCP 服务范围、默认工作流
2. **可视化工作流编排** — 支持 AI 生成、知识检索、条件判断、HTTP 请求、脚本执行、通知推送、MCP 调用、循环、并行、子流程等节点自由组合
3. **多租户架构（服务商版）** — 一套系统服务多个企业客户，每个租户独立后台、品牌、知识库、模型配置、工作流
4. **完整 RAG 检索链路** — 文档解析、切片、混合检索、重排、Query 改写、检索参数配置
5. **知识库管理** — 知识库/分类/标签/文件结构，支持多格式文档上传、网页导入、文件级管理、切片查看、知识范围绑定
6. **工具和 MCP 服务接入** — 后台可直接配置工具与 MCP 服务，智能体和工作流都能调用
7. **图文问答** — 聊天窗口支持上传图片，结合文字识别用户需求，模型自动识别文本/图文
8. **三端齐全** — 平台后台（服务商版）、租户后台、前台聊天页
9. **可观测可调试** — 聊天日志、请求日志、护栏事件、评测记录、工作流测试、统计报表
