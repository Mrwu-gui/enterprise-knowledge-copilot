# 运维排障

## 日常重点目录

- `data/app.db`：主数据库
- `data/tenants/`：租户配置
- `knowledge/`：知识文件
- `data/qdrant_store/`：本地向量数据
- `output/releases/`：导出结果

## 常见问题

### 页面打不开

- 查服务进程是否还在
- 查端口和反向代理是否正常
- 查对应前端页面文件是否被正确导出

### 登录失败

- 查账号是否存在
- 查对应租户是否启用
- 查认证头或登录页配置

### 问答没结果

- 查模型配置是否可用
- 查知识文件是否切片成功
- 查检索配置和日志里的命中情况

### 工作流报错

- 查节点输入输出
- 查外部接口返回
- 查脚本执行异常

### MCP 不生效

- 查服务地址
- 查鉴权信息
- 查开关是否启用

## 升级前必须备份

- `data/`
- `knowledge/`
- `.env` 和密钥配置
- 反向代理和进程托管配置

## 升级后重点验证

- 登录、聊天、后台页面是否正常
- 租户配置是否还在
- 知识检索和工作流是否还能跑
- 日志是否继续写入
