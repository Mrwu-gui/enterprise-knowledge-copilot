# 部署与启动

## 本地最快启动方式

```bash
pip install -r requirements.txt
python3 -m uvicorn backend.main:app --host 0.0.0.0 --port 6090 --reload
```

启动后常用入口：

- 登录页：`/login`
- 聊天页：`/chat`
- 企业版后台：`/tenant`
- 服务商版平台后台：`/admin`

## 初始化顺序

1. 安装依赖并启动服务
2. 配好模型和 API Key
3. 登录后台补齐品牌和登录提示
4. 上传知识文件并确认切片
5. 配智能体和工作流
6. 做问答测试和日志回看

## 生产部署建议

- 用 Nginx 做反向代理
- 用 `systemd`、`supervisor` 或容器托管进程
- 把 `data/` 和 `knowledge/` 做持久化
- 升级前先备份数据库、知识目录和配置文件

## 上线前验收

- 登录、聊天、后台入口都能打开
- 模型配置能保存并成功调用
- 知识文件能上传、切片、检索
- 默认工作流能正常执行
- 日志页能看到请求记录和聊天记录
